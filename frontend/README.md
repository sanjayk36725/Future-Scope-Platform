# Frontend Application

The frontend of the Future Scope Platform (FSP) is built using React 19, Vite, TypeScript, and Tailwind CSS.

## Structure
- `/src` contains the main source code including pages, components, and global state context.
- `/src/components` contains reusable visual widgets, buttons, forms, and cards.
- `/src/pages` contains the individual screens for each user role dashboard (Student, Faculty, Developer, Recruiter, HR Manager, Admin) and the specialized agent rooms.
- `/src/context` houses global configuration states, theme providers, and authentication states.
