# AI Agents & Models Framework

This directory governs all AI-powered services and specialized agents using the Google Gemini SDK.

## Structure
*   `/agents` - Individual agent definitions, context managers, and prompt overrides (Tutor, Code reviewer, Resume Screener, Mock Interviewer, HR Q&A, etc.).
*   `/services` - Core API clients interfacing with the Google GenAI SDK.
*   `/prompts` - Markdown-styled, robust system prompt templates for each agent to ensure strict persona guidance.
