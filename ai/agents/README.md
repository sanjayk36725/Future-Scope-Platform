# AI Agents Directory

This directory hosts the definitions, logic, and operational runtimes for all 25+ specialized AI agents configured in the FSP platform.

## Agent Domains
- **Education**: Personal Tutor, Coding Mentor, Assignment Helper, Quiz Generator
- **Placement Preparation**: Resume Review, Career Guidance, Attendance Analysis
- **Software Development**: Code Generation, Code Review, Debugging, Unit Test Generator, API Documentation, SQL Query Generator, DevOps Assistant, GitHub PR Review
- **Placement & Recruitment**: Resume Screening, Candidate Matching, Mock Interview, Skill Gap Analysis, Coding Evaluation, Communication Assessment
- **HR**: Employee Onboarding, Leave Management, Policy Q&A, Payroll Support, Employee Feedback
- **Smart Campus**: Student Helpdesk, Timetable Assistant, Lab Booking, Placement Coordinator, Event Registration, Library Assistant, Hostel Management, Fee Inquiry
